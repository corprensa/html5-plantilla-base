# html5-plantilla-base

Plantilla base para documento HTML5.


```html
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>Título de la página</title>
  <link rel="stylesheet" href="css/styles.css?v=1.0">
</head>
<body>
  <!-- Contenido de la página -->
  <script src="js/scripts.js"></script>
</body>
</html>
```
